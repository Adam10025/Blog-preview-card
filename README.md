## My process

### Built with

- Semantic HTML5
- CSS (`@font-face` for local Figtree fonts)
- Flexbox
- Mobile-first workflow

### What I learned

- How to center the card using Flexbox with `min-height: 100vh`
- How to use `box-sizing: border-box` so widths and padding are easier to reason about
- How to match typography and spacing to mobile and desktop JPG references
- How to load **Figtree** from local font files (weights 500 and 800)
- How to style a circular avatar with `border-radius` and `object-fit: cover`
- How to add hover and focus-visible styles for accessible interactive elements

### Continued development

- Fine-tune spacing and font sizes against the design files
- Keep practicing accessibility (focus, contrast, meaningful `alt` text)
- Explore CSS custom properties for colors and spacing on future projects
